HTML | CSS | Javascript

This is a simple and interactive implementation of the classic Tic-Tac-Toe game, built using HTML, CSS, and JavaScript. The project allows two players to take turns marking spaces on a 3x3 grid until one achieves a winning combination or the game ends in a draw.

  • Key features include:
  • A clean and responsive user interface.
  • Visual feedback for player actions.
  • Logic to detect wins, losses, and draws.

This project demonstrates core frontend development skills, including DOM manipulation, event handling, and basic game logic.

React | JavaScript | Firebase

This is an online food ordering platform built with React, using Firebase as the backend database. It enables users to browse a menu, manage a shopping cart, and submit orders.

This project showcases skills in React component-based development, state management (useState, useReducer), and Firebase integration, making it a great example of a basic e-commerce application.
